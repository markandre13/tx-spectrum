TX Spectrum (Work in Progress)

What

CSS for applications with a light and a dark look based on Adobe's Spectrum Design System and IBM's Plex font family.

Please note that TX Spectrum does not use the CSS provided by Adobe and if you find TX sucks, please blame me instead of Adobe.

Why

The browser is replacing the desktop and I was in need for a decent look to use in workflow and toad.js.

Adobe Spectrum is similar to the widely adopted Google Material.io, while a) reducing the amount of animation, b) reducing the amount of required DOM elements and c) covering the needs for application development.

IBM Plex replaces the Adobe Clean fonts, which are restricted for use in Adobe's applications, while Plex is free as in freedom.

CSS supports separation of concerns and independence from frameworks like WebComponents, Angular, React, Vue, Svelte, jQuery, ...
